# Navigation Bar
A simple tablet style navigation bar built using HTML, CSS and a little bit of JavaScript for switching themes.

# Overview
I created this project after learning the basics of HTML and CSS to test my practical understanding. The goal was to build something visually and functionally clean without using frameworks.

# Languages Used
- HTML - Structure
- CSS - Styling
- JavaScript- Theme switching

# Purpose
- Practice HTML and CSS concepts
- Basic understanding of HTML and CSS
